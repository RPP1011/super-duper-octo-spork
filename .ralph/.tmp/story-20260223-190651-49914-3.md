### US-004: Implement character creation step 1 (faction selection)
Status: in_progress
Depends on: US-002

Description:
As a player, I want to choose a faction after starting a new campaign so campaign context and bonuses are explicit.

Acceptance Criteria:
- [ ] New Campaign transitions to Character Creation faction screen before entering overworld.
- [ ] Faction choices display clear gameplay impact text before confirmation.
- [ ] Confirmed faction selection is stored in campaign state for subsequent steps and save/load.
- [ ] Example: selecting Faction-X applies expected faction identifier in campaign state and advances to backstory step.
- [ ] Negative case: attempting to continue without selecting a faction blocks progression with actionable UI feedback.
- [ ] `cargo test` passes with flow tests for faction selection persistence and gating.
