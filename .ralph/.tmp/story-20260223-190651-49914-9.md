### US-010: Bootstrap region-to-local eagle-eye intro sequence
Status: in_progress
Depends on: US-009

Description:
As a player, I want the local scene to start with my character exiting a dilapidated building so campaign immersion matches the intended narrative start.

Acceptance Criteria:
- [ ] Region layer can trigger local eagle-eye scene bootstrap with selected region context.
- [ ] Local intro places dilapidated building anchor and spawns player character initially hidden/inside.
- [ ] Intro sequence animates character exit and then hands control to normal gameplay input.
- [ ] Example: entering local intro from Region-D shows building anchor, exit animation, then controllable player state.
- [ ] Negative case: if anchor prefab/geometry is unavailable, bootstrap aborts safely with recoverable status and no hard crash.
- [ ] `cargo test` passes including tests for intro completion flag and input handoff state changes.
