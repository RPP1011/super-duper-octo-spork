### US-012: Add deterministic regression fixtures and screenshot stages
Status: in_progress
Depends on: US-011

Description:
As a developer, I want deterministic fixtures and screenshot baselines for key campaign stages so regressions in flow and visual state are caught early.

Acceptance Criteria:
- [ ] Deterministic fixtures cover start menu, character creation, overworld, region view, and local intro frame.
- [ ] Screenshot stage capture includes dedupe behavior and golden-baseline verification for the listed stages.
- [ ] Fixture execution under same seed yields stable state outputs required by regression assertions.
- [ ] Example: running regression fixture with seed S-001 produces matching baseline captures across repeated runs.
- [ ] Negative case: modified UI/state causing baseline mismatch fails regression checks with stage-specific diagnostics.
- [ ] `cargo test` passes including deterministic fixture assertions and screenshot-stage validation hooks.
