### US-005: Implement character creation step 2 (backstory/archetype effects)
Status: in_progress
Depends on: US-004

Description:
As a player, I want backstory selection with explicit effects so my campaign start identity changes gameplay outcomes.

Acceptance Criteria:
- [ ] Backstory screen lists archetypes with stat modifiers and recruit-generation bias details.
- [ ] Backstory confirmation writes modifiers to player/campaign state and marks character creation complete.
- [ ] Completion transitions into overworld map using same campaign seed.
- [ ] Example: selecting Scout archetype applies expected scouting/recruit bias modifiers and enters overworld.
- [ ] Negative case: invalid or missing backstory identifier cannot be committed and shows recoverable validation message.
- [ ] `cargo test` passes including tests that verify modifier application and completion flags.
